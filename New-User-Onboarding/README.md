# New-User-Onboarding

This source code allows the user to onborad a new hire quickly and efficiently

**What it does:** The script will one create the user in AD >> ask to mirror an existing user for security group provsioning >> send emails to all security group owners for approvals >> create an exchange online mailbox >> create VDI in vSPher servers and assign to the new user >> add the newly created VDI machine to the citrix machine catalog delivery group >> add SCCM application packaging access policy security groups in AD 

**Purpose:** The purpose is the help expedite, standardize and streamline the time it takes to completley onboard a new hire. Also reduces manual efforts
